A) HTML Structure - top

Below the default configuration:

$(document).ready(function () {
   $('#weather').Weatherflat(['USE ID WEATHER']);
});

The following example shows more detailed information of the current weather condition:

$(document).ready(function () {
	$('#weather').Weatherflat(['USE ID WEATHER'],{
		unit: 'f',
		image: true,
        photo: true,
		country: true,
		visibility: true,
		forecast: true,
	});
});
